# EPIC-02 Evidence: Baseline Architecture Design & Documentation

## Mục tiêu

Folder này lưu evidence cho EPIC-02: Baseline Architecture Design & Documentation.

1. AWS High-Level Architecture
2. EKS Namespace Application Architecture
3. External Services, Cost & Control Layer

Các sơ đồ này dùng làm source of truth cho deployment, performance baseline, cost baseline, observability và Week 1 Pitch.
